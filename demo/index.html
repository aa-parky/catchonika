Catchonika

Default-on MIDI capture. Lives happily in a tab.

About

Catchonika starts recording from all available Web MIDI inputs the moment this page loads. Save the last 60s or the entire session as a .mid.

  • Sustain (CC64) aware durations
  • Velocity preserved
  • Optional per-channel track export

Tip: Web MIDI requires a secure context (https or localhost) in most browsers.
